# UA3
Projet UA3

Projet UA3 – Gestion des étudiants

Membres

•	Badr Kabbadj
•	Safaa Namane

Description

Application Java permettant de gérer les étudiants, leurs cours et leurs notes à partir d’un fichier CSV.

Fonctionnalités développées

•	Lecture d’un fichier CSV
•	Calcul des moyennes
•	Tri des étudiants par moyenne
•	Ajout, suppression et recherche d’un étudiant
•	Sauvegarde des résultats

Instructions d'exécution

1.	Placer le fichier etudiants.csv dans le dossier du projet
2.	Compiler le projet avec : javac *.java
3.	Exécuter le programme avec : java Main
